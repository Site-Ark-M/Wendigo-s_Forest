# **Welcome to the Wendigo's Forest!**

This was a short solo project that I undertook as a capstone to one of my computing science classes during high school. It may be a bit clunky overall as one one my earlier projects, but it's got heart, and I continue to tweak things here and there.

You are free to download everything any anything! This should compile and run on any computer that can run Java programs. If you have any questions about redistribution/liscensing/the like, contact me.

*Keep in mind*, this project was completed without the use of classes, as that material had not been taught yet. I am working on a sequel in python that will be higher quality in this reguard.

## **RECOMMENDED WAY TO PLAY:**
- Download the .jar file.
- Using your system's command line interface (ie. Terminal for Mac, Command Prompt or Powershell for windows), find and run the jar file:
    java -jar "Wendigo's Forest.jar"

*Keep in mind that you will need a recent Java version installed to play this game.*

## **CONTROLS:**
- Standard WASD. Rapidly tapping is the fastest way to travel if you don't have key repeats (ttttttttttttthis) turned on
- e to shoot
- P to pause
- 1-4 to decide weapon at start
